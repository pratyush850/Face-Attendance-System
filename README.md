# Face-Attendance-System

A smart facial recognition-based attendance management system built using **Python**, **OpenCV**, **PyQt5**, and **SQLite**. The application automates attendance tracking by recognizing registered faces in real time, eliminating the need for manual attendance while improving accuracy and efficiency.

![Project Screenshot] (face.png)

## 🚀 Features

* 👤 Real-time face detection and recognition
* 📝 Automatic attendance marking
* 💾 SQLite database for storing student and attendance records
* 🖥️ User-friendly PyQt5 desktop interface
* 📷 Webcam-based attendance system
* ➕ Student registration with face dataset creation
* 📊 Attendance history and record management

## 🛠️ Tech Stack

* **Python**
* **OpenCV**
* **PyQt5**
* **SQLite**
* **NumPy**

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Face-Attendance-System.git
```

2. Navigate to the project directory:

```bash
cd Face-Attendance-System
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python main.py
```

## 📸 How It Works

1. Register a new user by capturing facial images.
2. Store facial data securely in the database.
3. Start the attendance module.
4. The system recognizes registered faces through the webcam.
5. Attendance is automatically recorded with date and time.
